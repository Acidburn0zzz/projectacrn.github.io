CONFIG_GPU_SBDF¶

(No prompt – not directly user assignable.)

Type: hex

Direct dependencies¶

DMAR_PARSE_ENABLED

(Includes any dependencies from if’s and menus.)

Defaults¶

  • 0x00000010 if DMAR_PARSE_ENABLED

Kconfig definition¶

At arch/x86/Kconfig:108, in menu (top menu):

config GPU_SBDF
    hex
    default 0x00000010 if DMAR_PARSE_ENABLED
    depends on DMAR_PARSE_ENABLED

(Definitions include propagated dependencies, including from if’s and menus.)
