.. title:: SOS_RAM_SIZE

.. option:: CONFIG_SOS_RAM_SIZE

*Size of the Service OS (SOS) RAM*

Type: ``hex``

Help
====

A 64-bit integer indicating the size of the Service OS RAM (MMIO not
included).


Defaults
========

- 0x200000000 if \ :option:`PLATFORM_SBL <CONFIG_PLATFORM_SBL>`
- 0x400000000 if \ :option:`PLATFORM_UEFI <CONFIG_PLATFORM_UEFI>`
