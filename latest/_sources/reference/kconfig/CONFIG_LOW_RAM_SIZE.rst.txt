.. title:: LOW_RAM_SIZE

.. option:: CONFIG_LOW_RAM_SIZE

*Size of the low RAM region*

Type: ``hex``

Help
====

A 32-bit integer indicating the size of RAM region below address
0x10000, starting from address 0x0.


Defaults
========

- 0x00010000

==================

.. highlight:: kconfig

Included via ``Kconfig:3``
