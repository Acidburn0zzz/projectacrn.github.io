.. title:: MAX_VCPUS_PER_VM

.. option:: CONFIG_MAX_VCPUS_PER_VM

*Maximum number of VCPUs per VM*

Type: ``int``

Help
====

The maximum number of virtual CPUs the hypervisor can support in a
single VM.

Defaults
========

- 8
