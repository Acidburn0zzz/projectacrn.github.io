.. title:: HYPERV_ENABLED

.. option:: CONFIG_HYPERV_ENABLED

*Enable Hyper-V enlightenment*

Type: ``bool``

Help
====

When set, the minimum set of TLFS functionality together with some
performance enlightenments are enabled.

Defaults
========

- y
