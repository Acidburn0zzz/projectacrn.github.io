.. title:: IOREQ_POLLING

.. option:: CONFIG_IOREQ_POLLING

*Polling mode*

Type: ``bool``

Help
====

When I/O request is completed, SOS will only mark completion status
without notifying hypervisor. Hypervisor will poll the completion
status and finish the post work.
