.. title:: IOMMU_BUS_NUM

.. option:: CONFIG_IOMMU_BUS_NUM

*Highest PCI bus ID used during IOMMU initialization*

Type: ``hex``

Help
====

Any BDF with a bus ID smaller than this number is mapped to
the IOMMU domain of the first VM.

Defaults
========

- 0x100

==================
