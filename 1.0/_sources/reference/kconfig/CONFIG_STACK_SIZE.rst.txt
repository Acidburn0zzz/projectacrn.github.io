.. title:: STACK_SIZE

.. option:: CONFIG_STACK_SIZE

*Capacity of one stack, in bytes*

Type: ``hex``

Help
====

The size of stacks used by physical cores. Each core uses one stack
for normal operations and another three for specific exceptions.


Defaults
========

- 0x2000

==================
