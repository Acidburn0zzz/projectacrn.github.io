.. title:: UOS_RAM_SIZE

.. option:: CONFIG_UOS_RAM_SIZE

*Size of the UOS RAM*

Type: ``hex``

Help
====

A 64-bit integer indicating the size of the user OS RAM (not included the MMIO).
Now we assume each UOS uses same amount of RAM size.


Defaults
========

- 0x100000000 if \ :option:`PLATFORM_SBL <CONFIG_PLATFORM_SBL>`
- 0x200000000 if \ :option:`PLATFORM_UEFI <CONFIG_PLATFORM_UEFI>`
